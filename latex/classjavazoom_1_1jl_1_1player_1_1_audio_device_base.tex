\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classjavazoom_1_1jl_1_1player_1_1_audio_device_base}{Audio\+Device\+Base}}} class provides a simple thread-\/safe implementation of the {\ttfamily \mbox{\hyperlink{interfacejavazoom_1_1jl_1_1player_1_1_audio_device}{Audio\+Device}}} interface. Template methods are provided for subclasses to override and in doing so provide the implementation for the main operations of the {\ttfamily \mbox{\hyperlink{interfacejavazoom_1_1jl_1_1player_1_1_audio_device}{Audio\+Device}}} interface.
