\doxysubsection{Detailed Description}
The {\ttfamily \mbox{\hyperlink{classjavazoom_1_1jl_1_1converter_1_1_converter}{Converter}}} class implements the conversion of an M\+P\+EG audio file to a .W\+AV file. To convert an M\+P\+EG audio stream, just create an instance of this class and call the convert() method, passing in the names of the input and output files. You can pass in optional {\ttfamily Progress\+Listener} and {\ttfamily Decoder.\+Params} objects also to customize the conversion.
