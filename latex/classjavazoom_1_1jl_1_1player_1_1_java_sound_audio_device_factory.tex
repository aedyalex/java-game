\doxysubsection{Detailed Description}
This class is responsible for creating instances of the \mbox{\hyperlink{classjavazoom_1_1jl_1_1player_1_1_java_sound_audio_device}{Java\+Sound\+Audio\+Device}}. The audio device implementation is loaded and tested dynamically as not all systems will have support for Java\+Sound, or they may have the incorrect version.
