\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classjavazoom_1_1jl_1_1decoder_1_1_java_layer_exception}{Java\+Layer\+Exception}} is the base class for all A\+P\+I-\/level exceptions thrown by Java\+Layer. To facilitate conversion and common handling of exceptions from other domains, the class can delegate some functionality to a contained Throwable instance.
