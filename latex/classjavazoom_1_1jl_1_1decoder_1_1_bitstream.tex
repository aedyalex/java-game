The {\ttfamily Bistream} class is responsible for parsing an M\+P\+EG audio bitstream.

{\bfseries{R\+E\+V\+I\+EW\+:}} much of the parsing currently occurs in the various decoders. This should be moved into this class and associated inner classes.
